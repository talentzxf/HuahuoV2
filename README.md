# HuahuoV2
Continue investigate the possibility to build web-ide

Please visit: www.huahuo.online.

Build:
1. Install emscripten (https://github.com/emscripten-core/emscripten) Note: Very complicated!
2. cd HuahuEngineV2
3. ./build.sh
4. cd ..
5. ./buildall.sh
6. cd HHIDE
7. npm run start
